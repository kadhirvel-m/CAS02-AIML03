How to Use Twitter Sentiment Analyzer

Analyze tweets using AI-powered insights in a few simple steps.

1
Enter a keyword or brand name in the search bar.
2
The system fetches relevant tweets using the Tweepy API.
3
Tweets are preprocessed to remove URLs and special characters.
4
RoBERTa analyzes the tweets and classifies sentiment.
5
The results are visualized with word clouds and charts.
